# Mobile-Classify
***
##数据集介绍

数据集包含2000*21维数据，最后一维数据为目标变量价格情况，0表示低成本，1表示中成本，2表示高成本，3表示非常高成本。
属性变量包含前20维，分别包含电池容量、是否有蓝牙、微处理器执行指令的速度、是否支持双卡、前置摄像头像素、后置摄像头像素等等属性变量。
